# Trailblazers Docs

This folder contains the durable documentation for the Trailblazers website project.

## Core working docs
- `STATE.md` — current project state
- `NEXT-STEPS.md` — active work queue
- `OPEN-QUESTIONS.md` — unresolved decisions
- `CHANGELOG.md` — major changes and rationale

## Architecture docs
- `ARCHITECTURE.md` — content model and system structure
- `SCHEMA.md` — field/schema conventions and implementation notes
- `TEMPLATES.md` — WordPress template plan
- `WORKFLOW.md` — development workflow
- `SYSTEM-INSTRUCTIONS.md` — persistent ChatGPT project guidance

## Operating rule
Treat these docs as the project’s durable memory. Update them when meaningful architectural, schema, workflow, or implementation changes occur.
